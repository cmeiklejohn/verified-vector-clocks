# vvclocks

Verified vector clocks, with Coq!

## Configuring

First, add a rebar dependency:

```erlang
{vvclocks, ".*", {git, "git://github.com/cmeiklejohn/vvclocks", {branch, "master"}}}
```

To run tests:

```erlang
make test
make itest
```
